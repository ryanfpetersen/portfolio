Part 1: What are semantic elements?

A web page is composed of distinct parts called semantic elements. These elements are written using HTML, and each has a specific meaning that defines its purpose to both the web developer and the web browser. For example, paragraphs, lists, headings and sections are all defined with semantic elements. Semantic elements are used to organize the web page and present the content in a standard, predictable manner. This standardized approach to creating web pages improves the accessibility and searchability of pages. It also improves consistency across the web. Semantic elements help the front-end developer create pages in a manner that improves the overall usability of the world wide web.

Learn more about semantic elements
